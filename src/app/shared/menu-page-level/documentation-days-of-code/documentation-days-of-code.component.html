About Days Of Code Tracker

Days Of Code Tracker Usage

When you click an element (circle button), it provides an option to enter notes. You do not have to enter notes.

The circle will turn green until it is clicked again and the notes will be cleared for that element.

Menu Options

  • View Goals: This is an option to toggle goals on and off. Goals can be added, edited, and deleted.
  • Export Blank Recordset: Export a JSON file without any data other than the 100 days of records.
  • Export Current Recordset: Export a JSON file with all the data you have saved for 100 days.
  • Import Saved Recordset: Overwrite the current recordset with something saved (CAUTION).
